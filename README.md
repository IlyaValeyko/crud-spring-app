## О проекте
CRUD Веб-приложение для управления пользователями.
## Основные операции
- Create - добавление нового пользователя
- Read - получения данных о пользователе
- Update - изменение информации о пользователе
- Delete - удаление пользователя
## Валидация данных
В проекте также реализована валидация для полей пользователя (name, age, email).
Ошибка валидации отображается красным цветом непосредственно под соответствующим полем.
## Технический стек
- Java 17
- Spring MVC 7.0.5
- HTML
- Maven
- Thymeleaf 3.1.2
- Hibernate Validator 9.1, Jakarta Validation API
- Apache Tomcat 11.0.18 (на версии меньше 10 не запустится)
